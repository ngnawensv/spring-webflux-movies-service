#### HTTP failures : Http failure falls in to categories
  1)  4xx - Client Error
  * 400 (Bad Request), 
  * 404(Not Found)
  * etc...

 2) 5xx - Server Error
 * 500(Internal Server Error),
 * 503 (Service Unavailable),
 * etc..
